diary20190105

发表于 2019-01-05 更新于 2020-12-14 分类于 程序人生

前几天立了几个flag，算是2019年的目标，2018年可以说是过得比较苦逼了，开始的时候算是用完了一年的运气吧，然后就很难受的过了大半年，最后的时候还算是找到了想做的事吧，运气这东西可靠不住，这一年，靠自己的努力，少打游戏，多做事。周围有很多的大佬，想成为大佬路途仍然遥远。

diary20181204

发表于 2018-12-04 更新于 2020-12-14 分类于 程序人生

中午放学回寝室的路上，听啊邱说他们几个都觉得跟我两个人一起走很尬，不知道路上该聊什么，我竟然一直都没发觉，感觉自己没救了。。跟女生不会聊天就算了，连跟男生聊天都不会了。。。。

13点58分突然觉得自己有点傻，做某件事要问问别人做不做，何必呢，人家都有自己要做的事，我为什么要等他们？明明已经把课内的东西丢的差不多了，搞课外再不搞出点成绩来，对得起自己吗?有些路，终究还是得自己一个人走，以后做事就干脆利落点，别人要不要弄是别人的事，自己想做就去做，努力学更多有用的东西。

diary20181114

发表于 2018-11-14 更新于 2020-12-14 分类于 程序人生

下午跟颜灿和刘浩钧在软院报告厅听了很多小组的英特尔杯项目，感觉创意很新颖的小组并没有几个，要么是自己之前想过的做过的，要么就是很早之前就听说过的了，感觉周围的人在创意点上并没有当时想象的那么强，还是得多关注关注生活，将所学用于生活。

emmmmmm。。。看到桂老师在群里说的要准备开始淘汰一部分人了，再想想自己这段时间以来学到的东西，怕不是得被淘汰哦。平时总是忙忙碌碌的，但真正学到的东西又有多少呢？恐怕很少吧，效率还是很重要的，接下来的学习，不要再把时间花在无谓的地方了，上课听不进去就别听，自己看书，听不进去还不做其他事，不就是浪费时间吗？上课效率还是很重要的，需要调整。

这段时间许多比赛陆陆续续地出来了，又有得忙了，但是，自己究竟能不能做的好呢？很多队友都看好我，认为我很强，但仔细反省反省自己，好像并不是这样，要学的东西很多，还有很多任务需要完成，得学会如何合理地安排时间了。

这段时间跟班里新转来的两个同学算是有点熟了，emmmmmm。。。让我比较想不到的是，刷绩点的居然会想着参加竞赛。。。而且还主动找的我，还是个妹子（似乎还对她有点点好感？？？这肯定不是我0.0），还参加两个比赛？？？？真的是。。想都想不到，大一一年了，除了大一上有妹子问过我一次之外就再也没有了，甚至也没什么人找我组过队，想想也是有点悲哀呢。但这次（可能还多亏了熊璟hhhh）。。。emmmmm。。加油吧，给人家留个好印象，再搞好关系，这样以后就又有可以能聊天的妹子了。
