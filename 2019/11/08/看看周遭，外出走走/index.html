看看周遭，外出走走

发表于 2019-11-08 更新于 2020-12-14 分类于 程序人生

这周宁哥给我们开了场班会，讲了讲他大学的一些经历。宁哥在问我们这两年多的大学生活是否有什么印象深刻的事的时候，我想了好久，似乎想不到什么。但听完他讲的几个经历后，才想起来我也有过类似的经历，但我却没法立马想起来。细细反思了下原因，可能是在这半年的学习中慢慢变得麻木了，基本每天都是上上课，到kaggle上学点东西，做做比赛，打打游戏，然后一天就过去了。于是乎，一时半会儿，没想起什么，后来仔细想想，是的，还是有蛮多印象深刻的事。

过去两年的日常，并不枯燥

大一国庆的时候，班导带着我们几个假期没回家的同学爬了岳麓山，虽说自己曾经在老家也经常爬山，但是当时确实被岳麓山的一些景色惊艳到了，其中印象最深的便是穿过一个很长的小道，再下几层台阶到的穿石坡湖

爬完岳麓山后，一行人便去了五一广场，吃完晚饭便向杜甫江阁走去，找了许久的空位，期待着在长沙看到的第一场烟花。

烟花结束后，回学校便成了一件很难的问题，路上都是人，车没法动，便只能走回去。当时一行人走在橘子洲大桥上，时不时还得注意边上经过的自行车和电动车，走了许久，在走到江对岸，运气很好，刚下桥没多久赶上了最后一班公交，回到宿舍也算是比预想的要早。

而后便是几天后与选哥（同级的老乡，软件专业，军训的时候认识的）一同去了世界之窗玩，那天天气很冷，出门特地穿了件外套，但还是顶不住长沙的妖风，又是国庆期间，人不是一般的多。差不多就是排队两小时，游玩两分钟，排队的时候两人被冻得瑟瑟发抖，那次也是我首次体验这些比较刺激的游乐设施吧。

在世界之窗里吹了一天的冷风的之后，两人决定去吃顿好的，于是去五一广场吃了顿自助。也忘了为啥，吃完后决定骑小黄车回宿舍，想想我还是在那个国庆假期刚开始的时候才学会的骑自行车，偶尔骑着骑着还得脚踩下地，都不知道当时自己哪来的勇气。然后回去的路，不是大上坡就是大下坡，简直刺激，在学期末跟几个好伙伴去五一广场搓完一顿回去的时候，我带他们也体验了一遍。不过这才两年，路都修好了，我也忘了当时那条路是怎么走来着了，真想再体验一波。

这之后还有许多蛮开心的事，一群人平时在寝室里打三国杀，期末考完一起出去聚餐；与一飞、磊哥在雪天上岳麓山；一个人去上海参加谷歌开发者节；信安作品赛国赛答辩结束后，一个人在南京玩了几天；与一群小伙伴去北京参加HackBJ，顺便去了趟清华；通宵刷ACM选拔题，第二天没补觉，跟一飞到星巴克继续刷题；参加百里毅行；脱过一次单。。。

外出走走，开拓视野

大一的时候，一直只是在学校里学课内的内容以及一些自己感兴趣的东西，直到大一下学期5月份的时候，参加了一场hackthon（因为之前在做ACM，罗大佬觉得我算法比较强就叫上了我 orz ），在那场hackthon中，第一次感受到了技术的魅力，那天晚上，学长刚开完毕业晚会就赶来比赛，做了一个通宵，我却做不了啥，只能在一边看着，偶尔在写法上给点优化的思路。最后在作品展示的时候，看到别人做的东西，意识到了与别人的差距。

去年11月份，跟几个伙伴商量好去参加上海的谷歌开发者节，结果因为期中考等原因，最后只剩我一人能去，考虑了很久，还是决定去。那时也是我第一回到上海，到的第一天晚上去外滩走了走，晚上睡觉前担心自己早上会起不来，结果一大早就醒了，到场时也已经有很多人了，发现大部分学生都是上海本地的，就我是大老远过来，真希望长沙到时候也能有个这样的活动。

还有今年4月份左右去北京参加的 HackBJ ，在去北京的那天还有课，开了张假条然后就跟小伙伴们去了，在去的路上，同学突然跟我说上课老师点名回答问题点到我了，那节课还有其他老师来听课，心里突然一慌，想起忘了跟老师讲了，于是等到下课的时间，发了条短信跟老师道歉，运气不是一般的背。。。到北京之后，跟在北京工作的学长学姐们约了一顿饭，大家一起聊了很多，收获满满。吃完在路上走的时候还看到了微软。

那天晚上回宾馆（青旅），对着百度地图找了好久，愣是没找着，后面打电话问，结果发现跟地图上的位置差了老远， 于是晚上12点，几个人还在街头走着，到了之后看到是这样的，卫生间也很小，澡都没法洗，emmmmm。。。。只能将就将就

然后就是第二天的hackthon，会场比预想的要小一点，不过也还行，最后也就拿了个计算资源的奖（我好菜啊 orz ）。。。。

还有前段时间去的世界计算机大会（好不容易遇上一个就在边上举办的），见识了不少东西，然后有些内容是英语讲的，只能靠看PPT理解，当时就应该去拿个同声传译器的。。

另外就是这近一年比赛的过程中，认识了不少大佬，在跟他们的交流中，不断地发现自己之前的见识面是多么的窄，懂得了我的竞争对手不应该是校内的同学，而是其他Top高校的大佬们，我们目前在学校里学到的东西跟他们比起来，差的不是一点。要不断变得更强！

这些给我印象最深的事，很少有与课内学习相关的。在大学刚开始的时候便想过，大学过的开心就好，不想再像初高中那样，整天坐在座位上，从早学到晚，不断刷题，要在玩中学。当时以兴趣选的专业，就以兴趣走到底，不想为了绩点把大学至少3年的时间花费在无意义的刷分上面。尽管有时会觉得当初自己做的一个不明智的选择就是放掉成绩，以至现在还在苦恼着能不能靠竞赛保研。但是，这两年的生活，收获很大。

最近也思考了以后的方向，大概有这么几个想法吧：

1. 竞赛保研，剩下的竞赛也不多了，感觉很难。
 2. 好好在实验室干活，发几篇paper，然后试着投一投MSRA。
 3. 直接出去工作，想做的方向大概有算法、大数据开发、分布式架构。

（突然就发现还有好多东西得学 orz ）

学点有意思的东西，打打比赛，认识些大佬，去外面见识见识，甚好。
