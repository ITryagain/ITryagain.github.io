diary20201018

发表于 2020-10-18 更新于 2020-12-14 分类于 程序人生

2020年10月18日23:30:48 今天完成了体侧补测，算了算成绩，应该是及格了。属实没想到，练了两年半的长跑，吃半年就没了。。。这个月恢复还算可以吧，稍微瘦了点，体能也恢复了挺多，好歹胖了20斤还是能跑进 4 分钟，虽然只有 3 分 58 - -。

这周还是有挺多内容没完成的，本来说要写的博客也还没开动，图形学光追的书也还没看完，下周得要抓紧时间；也差不多可以开始做毕设了，下次体测完差不多就要准备出去实习了，然后还有几件要做的事也得在出去实习前做了。

前几天跟学弟聊天的时候，学弟说到遇到一个志同道合的人很难，想一想确实啊，毕竟大学里大家的追求不一样。但仔细想一想，我似乎并没有搞明白自己追求的是什么，从大一的 ACM 到大二开始做 JAVA 然后紧接着就做机器学习和强化学习，一直到最后工作找的游戏开发方向，虽然一直都是以自己的兴趣为导向在做事，但是，我追求的到底是什么呢？感觉确实得花点时间去思考思考了，这个问题可能得在接下来几年生活中去解答了。
