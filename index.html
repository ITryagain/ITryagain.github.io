2020总结

发表于 2021-01-01 分类于 程序人生

2021年1月1日01:38:12 想了好久该怎么去写这篇总结，发现仍然想不出，那就还是想到啥就写啥吧。

这篇文章既是过去一年的一个总结，也是对未来一年的期待

写文章前，看了看去年上半年写的几篇日记、总结，发现原来自己都忘了好多东西，原来当初还立了那些 flag。

  1. 争取拿到竞赛保研资格，从高中到现在，打竞赛也打了三四年了，但是几乎没有获得过满意的成绩，不知是自己水平就是如此，还是就差那么点运气。所以这次，不再指望运气，说做到最好就做到最好，不划水，竭尽全力。毕竟自己当年放弃了绩点，但确确实实是想做几年的科研，可能会很累，但是真的想！虽说如此，仍然还没有考研的想法。
  2. 维护一个开源项目，或是 github 上已有的，或是自己做的。
  3. 做更多的分享，写更多的高质量博客。
  4. 向喜欢的人告白，这件事想了很久了。对我来说，把喜欢说出口真的好难。总是有所顾虑，或许她已经明白了我的想法了，如果真是这样，那被拒绝的概率可以说是接近 100% 了吧。可是一直憋在心里真的是难受，而且，现在感觉自己真的有点像是工具人吧。
  5. 发几篇水平比较高的论文，即使后面拿不到竞赛保研资格，发了几篇论文，或许能靠这些论文来申请 MSRA 或是一些 AI lab。
  6. 每天或每周抽点时间看看 CSAPP 和 SICP 或是一些技术方面的书籍，就算搞算法，工程能力还是得要有。

6个 flag 最终只做到了两，实际上第 3 个 flag 也不算完成的很好，毕竟从 6 月份开始就没写博客了，断更了半年 orz，想想粉丝数还在不断地涨就感觉有些小尴尬。

而其他未实现的 flag，仔细一看，只能说，突如其来的疫情，打乱了我当时的“计划”。最终走上了一条截然不同的路，不过想一想总的来说也并不算太差，完成了进大学时的目标（网易 offer），开始走上了游戏开发的道路。前天收到的网易元旦包裹里的那本书封面上有句话很喜欢，“筑梦之路，造物工程”，回想起大一时候思修课上，老师曾让我们写自己的梦想还是什么来着，当时具体写的内容记不太清了，但大致是“做出能够惊艳世界的游戏”，当然，这肯定是很久以后的事了。

过去这一年也是初中以来，跟家里人呆一块最久的一年了，将近半年，但仔细想了想，好像半年时间，大多时间也都是在做自己的事，跟家里人说的话比以前少了好多好多。前几个月家里好像还出了什么事，爷爷奶奶叫爸妈别跟我说，怕影响我学习。啊，是啊，从初中回老家读书开始，基本上每年能跟家里人在一块的时间就是寒暑假了，仍记得初中时对寒暑假是多么的期待，回到家跟爷爷奶奶总有说不完的话。然而，随着成绩逐渐变好之后，跟家里人聊的更多是学习、成绩，其他的就越来越少了，同时自己也变得越来越不会聊天。现在回到家，有时候在那看一天电脑，爷爷奶奶也不太敢来打扰。暑假两个半月的实习，认识了不少热爱游戏的朋友，也算是头一次体会了工作，不过实习期间实习生都不加班，自然是挺轻松的，周末跟学长们出去恰恰火锅，聊聊天。感觉正是这段时间，让我有所改变，“能把事情做完、做好，实现自己的目标，能够问心无愧就好了，那么有时候拼命的学习、干活究竟是为何？自己又到底在追求什么呢，现在想的是要好好体验生活，做做自己想做的事，最重要的还是要过得快乐才是啊。”

过去一年，认识了很多人，也经历了不少事，大体上还是过得还算比较满意，接下来的一年，再给自己立些 flag 目标吧：

  1. 做好目前手头上的项目，好好把团队做起来
  2. 继续做更多的分享，写更多的高质量博客，把知乎的新专栏做起来
  3. 做些游戏开发相关的视频，投到 B 站
  4. 减肥，毕业前减到 62kg 左右（2020年体重变化 65 -> 71 -> 75 -> 68)
  5. 脱单，ball ball 了，最后半年了，给个脱单的机会吧 orz
  6. 继续开源项目的建设
  7. 想到了再写吧

2021年1月1日03:03:17 没写多少东西，就 3 点了，感觉还有不少东西可以去写，就之后慢慢补上吧，新的一年，继续加油。
